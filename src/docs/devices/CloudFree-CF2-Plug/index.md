## GPIO Pinout

| Pin    | Function                     |
| ------ | ---------------------------- |
| GPIO0  | Status LED - Red (inverted)  |
| GPIO12 | sel_pin hlw8012 (inverted)   |
| GPIO5  | cf_pin hlw8012               |
| GPIO4  | Relay                        |
| GPIO13 | Button (inverted)            |
| GPIO14 | cf1_pin hlw8012              |
| GPIO2  | Status LED - Blue (inverted) |
